# JL Solutions Automation Portfolio

Welcome to the JL Solutions Automation Portfolio website! This site showcases the technologies, frameworks, and tools that power modern automation, DevOps, and AI-driven productivity. Built as a personal portfolio, it highlights expertise in automation engineering, test management, CI/CD, and intelligent assistants.

## Purpose
- Demonstrate hands-on experience with leading automation and DevOps tools
- Share benefits and use cases for each technology
- Present a professional profile and contact information
- Inspire teams and individuals to adopt automation and AI solutions

## Features
- **About Me:** Background and philosophy on automation and quality engineering
- **Frameworks & Tools:**
  - UI Testing: Selenium, Selenide, Playwright, Cypress, TestNG
  - API & Performance: Postman, JMeter, MCP
  - DevOps & Observability: Docker, Maven, Datadog, GitHub, GitLab, Jenkins
  - Test Management: TestRail, Jira, Confluence
  - AI & Intelligent Assistants: ChatGPT, GitHub Copilot, Amazon Q
- **Contact Section:** Email, phone, website, location, and social media links
- **Modern Design:** Responsive layout, dark/light theme toggle, SVG icons

## How to Use
- Browse the site to learn about each tool and its benefits
- Click "Learn more" links for official documentation
- Use the contact section to reach out for consulting, collaboration, or opportunities

## Technologies Used
- HTML5 & CSS3
- Responsive design principles
- SVG icons
- No backend required (static site)

## Getting Started
1. Clone or download the repository
2. Open `index.html` in your browser
3. Customize content or styles as needed
